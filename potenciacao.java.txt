/*
 *A potenciacao serve para simplificar calculos com numeros extensos, e em programacao isso é muito mais simplificado, apenas utiliza-se a funcao. Uma outra forma para realizar calculos dessa natureza é utilizando o laço de repeticao for através de um metodo que tem como parâmetros valores LaTeX:{double} e{int}.

Com base na informacao acima, realize o calculo da potenciacao utilizando o laço de repeticao {for} conforme as orientacoes abaixo:

- Escreva um metodo em que receba em seus parametros um valor {double,x} e um inteiro  {y};

- O metodo deve retornar um valor double x^y

- Considerar o valor y >=0.

De acordo com as recomendacoes acima, o codigo correto para o calculo da potenciacao e:
 */
package potenciação;

/**
 *
 * @Marcellopro
 */
public class Potenciação 

{
    static double potencia (double x, int y){
        double resposta = 1;
        for(int i=0;i <y; i++){
            resposta *= x;
        }
        return resposta;
        
    }
    
            

    /**
     * @param args the command line arguments
     */
    public static void main(String[] args) {
        System.out.println (potencia(23,50))
        
    }
    
}
